Software Administrador de Basculas WLS 

Administra y configura las basculas modelo WLS 
de una forma practica y sencilla.


1.- Haga click en Admin WLS v1_3.zip para extraer la informacion.

2.- Ejecute el archivo setup.exe para iniciar la instalacion.

3.- Haga click en el icono Admin WLS en el escritorio del equipo.

4.- La aplicacion pediras un usuario y contraseña solo ponga Admin en el usuario y de aceptar.

5.- La aplicacion tiene 7 modulos que son:
	BASCULAS - se dande alta las basculas
	CATALOGOS - se da de alta toda la informacion necesaria como datos maestros.
	SICRONIZACION - se envia los datos que requiera necesarios para operar a las basculas.
	CAMBIO DE PRECIO - se envia los cambios que posteriormente tenga en su negocio a las basculas.
	CORTE DE CAJA - se obtiene las ventas de los cortes de caja de las basculas para ser analizados.
	ESTADISTICAS - son todos los reportes para ver la infromacion extraida de las ventas.
	MANTENIMIENTO - es para configurar el sistema, administrar usuario y respaldar la base de datos.

Usted puede iniciar la operacion y el uso de sus basculas administradas a travez de esta aplicacion.
